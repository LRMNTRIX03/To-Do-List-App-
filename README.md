# To Do List App with React Ts
Aplikasi **To Do List** ini dibangun menggunakan **React dengan TypeScript (React.ts)** untuk membantu pengguna mengorganisir berbagai tugas dan aktivitas sehari-hari dengan lebih terstruktur. Aplikasi ini dirancang dengan tampilan yang modern serta antarmuka yang **user-friendly**, sehingga mudah digunakan oleh berbagai kalangan pengguna. Dengan fitur pencatatan dan pengelolaan tugas yang sederhana namun efektif, aplikasi ini dapat membantu meningkatkan produktivitas serta memastikan setiap pekerjaan dapat terpantau dan terselesaikan dengan baik.

# Cara Penggunaan 
1. **Clone Project**
```bash
git clone https://github.com/LRMNTRIX03/To-Do-List-App-
```

2. **Masuk ke Folder**
```bash
cd todo-list-app
```

2. **Jalankan serverr**
```bash
npm run dev
```
